# WATI Codex — Notice

WATI Codex consumes metadata supplied by WATI Core. It does not include third-party add-ons or third-party textures.

All icons in the WATI Codex resource pack are original generic fallback symbols. They indicate broad content categories and do not reproduce the artwork of the referenced add-ons.

Third-party project names and identifiers shown by the interface are used for identification and interoperability. See the WATI Core `CATALOG_POLICY.md` and `THIRD_PARTY_SOURCES.md` files.
